# 🐖 Swine Breeding Manager - Premium Management System

Một hệ thống quản lý trang trại heo giống chuyên nghiệp với giao diện **Glassmorphism** hiện đại, được tối ưu hóa cho trải nghiệm người dùng tinh tế và hiệu quả.

---

## ✨ Điểm Nổi Bật (V.I.P Features)

### 🎨 Giao Diện Glassmorphism Đẳng Cấp
- Thiết kế **nền mờ (Backdrop Filter)** sang trọng.
- Hiệu ứng Gradient động và các thành phần giao diện "phát sáng" hiện đại.
- Tối ưu hóa hiển thị trên mọi thiết bị (Responsive Design).

### 🧬 Quản Lý Gia Phả Chuyên Sâu
- Cây gia phả (Genealogy Tree) trực quan, hiển thị toàn diện các thế hệ: Bố mẹ, Anh chị em và Đời con.
- Định danh chính xác qua **Số Tai (Tag Number)** và **Tên riêng**.
- Phân loại màu sắc giới tính thông minh (Anh - Xanh, Chị - Hồng).

### 🏗️ Sơ Đồ Chuồng Trại Trực Quan
- Quản lý heo theo từng chuồng cụ thể.
- Tính năng **Tự Động Phân Chuồng** cho heo con mới đẻ.
- Theo dõi mật độ nuôi và tình trạng chuồng trại theo thời gian thực.

### 🛡️ Phân Quyền Cá Nhân (Dynamic Permissions)
- Hệ thống bảo mật đa tầng: **CHỦ TRẠI** và **CÔNG NHÂN**.
- Admin có thể trùy chỉnh quyền truy cập **chi tiết đến từng trang** cho từng nhân viên cụ thể.
- Trang lỗi **404 Premium** cực kỳ "cute" và thân thiện khi có truy cập trái phép.

### 📈 Báo Cáo & Thống Kê Thông Minh
- Biểu đồ tăng trưởng, doanh thu và thống kê sinh sản chuyên nghiệp.
- Tự động tính toán ngày dự kiến sinh và nhắc nhở lịch phối giống.

---

## 🚀 Công Nghệ Sử Dụng

- **Backend**: ASP.NET Core 10.0 (Latest)
- **Database**: Entity Framework Core với SQLite (Nhẹ, nhanh, cơ động)
- **Frontend**: Razor Pages, Vanilla CSS (Custom Glassmorphism), JavaScript
- **Security**: ASP.NET Core Identity & JWT
- **UI Libraries**: FontAwesome, SweetAlert2, Google Fonts (Inter/Outfit)

---

## 🛠️ Cài Đặt & Khởi Chạy

1. **Yêu cầu**: Cài đặt .NET 10.0 SDK.
2. **Clone project**:
   ```bash
   git clone <link-github-cua-ban>
   cd SwineBreedingManager
   ```
3. **Cập nhật Database**:
   ```bash
   dotnet ef database update
   ```
4. **Chạy ứng dụng**:
   ```bash
   dotnet run
   ```
   Sau đó truy cập `http://localhost:5281`
